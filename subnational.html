What the Data Reveals

Between 2010 and 2024, Infant Mortality Rate across Europe has improved by 0.1% on average — but the average conceals a stark gap.

In 2024, a person in Åland (Finland) faced outcomes scored at 148.6, while someone in Východné Slovensko (Slovakia) faced 25.2.

Today, 95 regions still fall below the EU average — each one representing populations whose health outcomes lag behind their neighbours.

Beyond Europe — Inequality Is Everywhere

This demonstration uses European health indicators, but the same approach applies wherever populations face unequal outcomes. How the Evidence Is Built

Every number on this map is produced by localintel, an open-source R package designed for transparent, reproducible subnational analysis

Direct from Official Sources

Indicators are retrieved programmatically from Eurostat's bulk download facility through the official API — the same data that underpins EU Cohesion Reports and national policy reviews. The retrieval is parameterised by indicator code, NUTS level, and time range, so every query is auditable and reproducible.

get_nuts_data(indicator, nuts_level = 2) localintel · get_nuts_data

Cross-Border Harmonisation

NUTS boundary codes change between classification revisions (2013, 2016, 2021), breaking time-series comparability. The package maps regions across revisions automatically, so a 15-year trend line for a Portuguese region and a Polish one are measured on exactly the same terms.

harmonize_nuts(data, from = "NUTS2016", to = "NUTS2021") localintel · harmonize_nuts

No Region Left Blank

Subnational data is structurally incomplete — some regions report, others don't. Where values are missing at NUTS-2, the package cascades upward through the geographic hierarchy (NUTS-1, then national), ensuring every region appears on the map and no community is invisible to policy.

coalesce(nuts2_value, nuts1_value, nuts0_value) localintel · cascade imputation
